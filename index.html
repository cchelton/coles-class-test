My JS class test

I wanted to try to make a class counter that incremented whenever a new class object was constructed. This is what I came up with. See test.js

There are 3 classes

  • Vehicle: Parent class, increments whenever a Vehicle, Bike or Car is constructed
  • Bike: Extends from Vehicle. Has its own counter but increments Vehicles as well
  • Car: Same as Vehicle

All 3 classes have a static field that can be referenced, saving me the time of writing a getter method for the count
